# Single-purpose statement

> Required by Chrome Web Store. Paste verbatim into the "Single purpose"
> field. Keep under 1000 characters.

AllyProof scans the current web page for accessibility (WCAG 2.2 AA)
violations and reports them to the user. It runs the open-source
axe-core engine locally in the user's browser, surfaces the issues
found, and offers an optional sign-in flow to save scans, generate AI
fix suggestions, and crawl the entire site through the AllyProof
dashboard.

Every feature in the extension serves this single accessibility-audit
purpose: the popup shows the score and severity counts, the side panel
lists the violations, the badge shows the issue count, and the "Show
on page" buttons highlight the failing elements in place. There is no
unrelated functionality.
